第一章：准备知识

集合

基数：实际上就是有个几个不同的元素，代表几元。$ |A|$

幂集：一个集合的所有子集构成的集合就是其幂集 。$P(A) 或者 2^A$ 。 如果该集合为n元则其幂集元素共有2的n次方个。

对称差：$A \oplus B = (A-B) \cup (B-A) $ A和B除去交集的区域。

整数

整除：$a|b , b = qa+r$ . q为商（Quotient）,r称为余数（Remainder）

推论1： $ a|b ,a|c => a|(mb+nc)$

素数（质数）：只能被自身和1整除的正整数（不包括1），反面称为合数。

定理1.3 ： 素因子分解：每个大于1的整数n，能够被分解成唯一的素因子序列。

定理1.4：每个合数必有一个小于或等于$ \sqrt{n}$ 的素因子

是否是素数的有效算法：

gcd：Greatest Common Divisor 最大公约数。

互素：两个整数最大公约数为1时，称两数之间互素

两两互素：一列整数互相直接都是互素的，每两个直接的gcd都是1。

gcd的素因子分解求法：两个分解为两个素因子序列，将素因子的指数设定为两者相同素因子的指数中较小的一个，之后相乘。

定理1.5：如果a,b 和d时任意的正整数，d = gcd(a,b) ,则

  • 存在整数s,t ,使得d = sa+tb
  • 如果c是a 和b 的公因子，则c|d

证明思路：

令c是a和b的公因子;

$c|a,c|b \rightarrow c|x 且c<=x$

a = qx+r 反表示r

之后x用sa+tb表示

整理式子，让r为a,b各自某一个倍数之后的和。

与x是a，b的某一个倍数和最小的正整数的事实矛盾。推出r一定是0，x|a。同样的方式证明x|b。

lcm：Lowest Common Multiple 最小公倍数

定理1.6：$gcd(a,b)lcm(a,b) = ab$

欧几里得算法求gcd（辗转相除法）：

  • A > B
  • R = A%B
    • if R == 0 : B is gcd(A,B)
    • if R! = 0: A = B,B = R
  • Execute again R = A%B

模运算：

只关心余数

a,b模m同余 ：$ a≡b (mod m) \leftrightarrow a = b+km $

序列和递推关系

序列：按某种顺序排列的一张表，有限个数为有限序列，无限个数则是无限序列。{a~n~}

等比序列： $ a_n = a_1 * q^{n-1}$

正负1交替序列： $ a_n = (-1)^{n+1}$

一般交替序列： $ a_n = \frac{a+b}{2}+(-1)^n\frac{b-a}{2}$

序列求和：$\sum\limits^{n}_{i=1}{a_i}$

序列求积：$\prod\limits_{i=1}^{n}{a_i}$

求和公式：

  • $\sum i = \frac{n(n+1)}{2}$
  • $\sum i^2 = \frac{n(n+1)(2n+1)}{6}$
  • $\sum i^3 = \frac{n^2(n+1)^2}{4}$

递推关系：一个表达序列中前后两项关系的等式

矩阵

布尔矩阵：所有元素都是0或者1的矩阵

  • 并：$ \vee$ 表示每个布尔值之间取 或，为什么并是或，因为或实际上放松了条件，增加了覆盖的面积
  • 交：$\land$ 表示每个布尔值之间取 和。和让条件更加严格，减小了面积。
  • 布尔积：就是矩阵乘法，但是将加法用$\vee,or$ 来代替，乘法用$\land ,and$ 来代替

第二章 数理逻辑

命题与联结词

命题： 陈述句并且有确定的真值

命题联结词：

  • 否定式：$\urcorner$ 取反
  • 合取式：$\land $ 为and
  • 析取式：$\vee$ 为or
  • 蕴含式：$\rightarrow$ 如果，则；只有前面为真，并且后面为假时，才能指定该命题为假
  • 等价式：$\leftrightarrow$ 当左右的真值相同时，才为真
  • 运算优先级：否定 > 合取 > 析取 > 蕴涵 > 等价 ，同级时按从左到右

命题符号化（难点）：

  • 只有……才
  • 仅当
  • 除非
  • 当且仅当

命题公式和分类

命题常项：不可再分的陈述句

命题变元（命题变项）：陈述句的组合或者变式

合式公式：单个命题变项是合式公式，有限次的应用联结词的符号串才是合式公式。

合式公式的层数：单个命题变项称为0层，之后每一级联结词加一层，出现合取析取等价蕴含的时候去两侧层数大的一层加一。

命题公式的赋值：给命题符号指定一个值，最终使得这个命题为真的数值就是成真赋值

真值表：就是将每个命题符号的可能写出来，然后将当中每个子项的真值写出来，最后写出完整命题的值

永真式（重言式）：无论怎么赋值都使得命题为真

永假式（矛盾式）：无论怎么赋值都使得命题为假

可满足式：只要不是矛盾式就是可满足式，可满足就是至少有一种情况能使其为真。

等值演算和范式

等价：两个形式的命题公式具有相同的真值，就称其为等价。记为：$\Leftrightarrow$

判断两个命题是否等价可以使用列出真值表比较的方法。

常用等值式：

  • 双重否定律：$A \Leftrightarrow \urcorner \urcorner A$
  • 幂等律：$A \Leftrightarrow A \vee A$ , $A \Leftrightarrow A \land A$
  • 交换律：
  • 结合律：
  • 分配律：$A\vee (B \land C) \Leftrightarrow (A\vee B) \land (A \vee C)$
  • 德~摩根律
    • $\urcorner (A \vee B) \Leftrightarrow \urcorner A \land \urcorner B$
    • 取反符号放入括号内，主取析取要改变
  • 吸收律
    • $A \vee (A\land B) \Leftrightarrow A$
    • $A \land (A\vee B) \Leftrightarrow A$
  • 支配律
    • $A \vee 1 = 1$ or 1 一定是 1
    • $A \land 0 = 0$ and 0 一定是 0
  • 同一律
    • $A \vee 0 = A$
    • $A \land 1 = A$
  • 排中律
    • $A \vee \urcorner A \Leftrightarrow 1$
    • 0 or 1 肯定是1
  • 矛盾律
    • $A \land \urcorner A \Leftrightarrow 0$
    • 0 and 1 肯定是0
  • 蕴涵等值式
    • $ A \rightarrow B \Leftrightarrow \urcorner A \vee B$
    • A 推出B，等价于，非A或者B
  • 等价等值式
    • $A \leftrightarrow B \Leftrightarrow (A\rightarrow B) \land ( B \rightarrow A)$
    • AB互推，就是A可以推出B 并且 B也可以推出A
  • 假言易位：
    • $A \rightarrow B \Leftrightarrow \urcorner B \rightarrow \urcorner A$
    • 逆否
  • 等价否定等值式
    • $A\leftrightarrow B \Leftrightarrow \urcorner A \leftrightarrow \urcorner B$
    • AB互推，非A和非B也可以互推
  • 归谬论
    • $(A\rightarrow B) \land (A \rightarrow \urcorner B ) \Leftrightarrow \urcorner A $
    • A推出B并且A又能推出非B，和非A是等价的。
    • 反证法

置换规则：使用前面的等值式来把左侧变换成右侧。

范式：

文字：$ q$, $\urcorner q$

简单析取范式：有限个文字构成的析取式 ： $p\vee q \vee \urcorner p$

简单合取范式：有限个文字构成的合取式 ： $p\land q \land \urcorner p$

析取范式：有限个简单合取范式构成的析取式 : $(\urcorner r \land p)\vee (q \land p)$

合取范式：有限个简单析取范式构成的合取式 : $(\urcorner r \vee p)\land (q \vee p)$

范式存在定理：任何一个命题公式 都存在一个与之等价的析取范式与合取范式

析取范式的步骤：

  • 将每个$\rightarrow ,\leftrightarrow $ 都消除
  • 将$\urcorner$ 移动到命题变元的前面
  • 运用分配律产生简单合取式或简单析取式（求析取范式时要产生简单合取范式 ，合取范式反之）

范式不是唯一的。

极小项：极小项对应简单合取式， 当每个命题变元的本身和其否定，只出现其中一个，并且每个命题变元按照角标或者字典正确排序时称该简单合取式为极小项。（意思就是已经转换到最短了。）

主析取范式：由有限个极小项组成的析取范式。（注意：极小项是简单合取式）

主析（合）取范式的存在唯一定理：任意一个命题公式都存在着与之等值的主析取范合和主合取范式，并且是唯一的。

主析取范式构造步骤：

  • 求析取范式
  • 若其中简单合取式不含某个变项，用分配律变成两个简单合取式
  • 排序去掉重复的简单合取式

极小项和极大项的性质：

  • 矛盾式不含任何极小项（只能用0来表示），永真式含有所有2^n^ 个极小项
  • 永真式不含任何极大项（只能用0来表示），矛盾式含有所有2^n^ 个极大项

主析取范式的应用：

  • 判断两个命题公式是否等值
  • 判断命题公式的类型
  • 求出公式的成真赋值和成假赋值
  • 可分析和解决实际问题

命题逻辑的推理理论

推理是有效的判断：列出前提合取式和结论的真值表，如果出现前提合取式为真，但是结论为假的情况，则推理无效。

推理有效的符号：$\Rightarrow$

前提合取式：就是将前提给出的所有条件合取。

有效推理的充要条件：$A_1 \land A_2 \land A_3 .. . \rightarrow B$为重言式。

演绎法证明推理：

推理规则：

  • 前提引入规则：在证明的任何步骤之前都可以引入前提
  • 结论引入规则：在证明的任何步骤上得到的结论都可以引入作为后继证明的前提
  • 置换规则：在证明的任何步骤上，命题的子公式可以用等值的公式置换

推理定理：

  • 附加律：$A \Rightarrow (A\land B)$

  • 化简律$ A\land B \Rightarrow A $

  • 假言推理 $ A\rightarrow B ,A \Rightarrow B$
  • 拒取式$A\vee B,\urcorner B \Rightarrow \urcorner A$
  • 析取三段论$A\vee B , \urcorner B \Rightarrow A$
  • 假言三段论$ A\rightarrow B ,B\rightarrow C \Rightarrow (A\rightarrow C)$
  • 等价三段论: $A\leftrightarrow B,B\leftrightarrow C \Rightarrow A \leftrightarrow C $
  • 构造性二难 $ A\rightarrow B,C\rightarrow D,A\vee C \Rightarrow (B\vee D)$
  • 合取律 $A,B \Rightarrow A\land B$

附加前提证明法：使用结论中的前件作为推理的前提，推出后面的。

谓词逻辑基础

个体词：指可以独立存在的客体，可以是一个具体事物，也可以是一个抽象的概念

谓词：一般用来指明客体性质（属性）或者客体之间的关系

谓词常项：具体的性质和关系

谓词变项：抽象、泛指的性质和关系

n元谓词：含有n个个体变项的谓词。

0元谓词：不带个体变项的谓词

0元谓词常项就是命题（注意：谓词不是命题，即使式谓词常项也不一定是命题，只有当该谓词常项的个体变元消失的时候才是命题）

谓词可以看作是一个函数，有几个变量就是几元谓词，当变量被赋值了之后就变成了0元谓词。

量词：量词是用于指明当前个体域数量的。

全程量词：$\forall xF(x)$

存在量词：$\exists x F(x)$

加上了量词的谓词就成为了命题。

个体域和全总个体域的符号化结果是不同的，全总个体域的情况下需要自己补充限制条件。

合式公式：

  • 原子公式是合式公式
  • 可以由合式公式组合

$\forall x A$ :

  • x是指导变元
  • A是相应量词的辖域
  • A中x的所有出现都是约束出现，称为约束变元
  • A中除x之外的所有出现都是自由出现，称为自由变元

改名规则：

  • 将量词中出现的变元以及该量词辖域中此变量的所有约束出现，都用新的个体变元替换。
  • 新的变元一定要有别于改名辖域中的所有其他变量。

代入规则：

  • 将公式中出现该自由变元的每一处都用新的个体变元替换。
  • 新变元不允许在原公式中以任何约束形式出现。

闭式：A是一个任意的公式，A中没有自由出现的个体变元，则称A为封闭的公式，简称闭式

谓词逻辑中公式A的一个解释I:

  • 非空的个体域集合D。
  • A中的每个常量符号，指定D中的某个特定的元素。
  • A中的每个n元函数符号，指定Dn到D中的某个特定的函数。
  • A中的每个n元谓词符号，指定Dn到{0,1}中的某个特定的谓词。

谓词公式的类型：永真式（逻辑有效式）、永假式（矛盾式）、可满足式

代换实例：设A0是含有命题变项p1, p2, …, pn的命题公式，A1, A2, …, An是n个谓词公式，用Ai（1<=i<=n）处处代换A0中的pi，所得公式A称为A0的代换实例。

重言式的代换实例是重言式，矛盾式的代换实例是矛盾式。
